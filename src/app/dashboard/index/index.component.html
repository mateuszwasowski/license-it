License It is a license managment system for developers.

It provides all the essential tools, to help you

govern your applications users and clients.

1

Register and create a group.

Creating a group gives you access to your very own group site. From there you can add applications to it and manage users and clients. You can also join an existing group if you got an invitiation. Don't worry! Your account can create and join multiple groups.

2

Connect License It with your application via our API.

Go to our Docs site to help guide you through the setup process. We need this connection in order to verify your clients licenses, check if they haven't expired and block them access to your app if needed.

3

Add clients and their licenses and watch their stats!

Now you can add clients and their licenses. You can deactivate licenses as well as schedule expiration dates and many more. A worthy note to add is that one client can have multiple licenses!
