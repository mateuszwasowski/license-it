Introduction

Welcome to the LicenseIt API! You can use our API to connect yourself to LicenseIt endpoints, which can get information about licenses in your applications.

You can view endpoint responses to the right, and the description of each request on the left. Let us know if you need any help setting up!

If you wish to see other code examples (Ruby, Javascript, other), feel free to visit this: Public Api Docs

Check a specific license

This endpoint retrieves info about a specific license

HTTP Request

POST licenseit.pro:5001/api/License/Check

BODY Parameters
License NumberLicense key you wish to access
Identity NumberThis is your personal identifier key
Application HashThis is your application secret in which the license is registered. The secret key can be found in the main application view.

Please note that all request should have content-type: application/json

        When license is incorrect:
        {{"\{"}}
            "isActive": false,
            "assignedVersion": 0,
            "expiration": null,
            "description": "No active license",
        {{"\}"}}
      
        When license is correct:
        {{"\{"}}
            "isActive": true,
            "assignedVersion": 1,
            "expiration": "2018-04-12T17:17:48",
            "description": "License Active",
        {{"\}"}}

Activate a license

This endpoint retrieves info about a specific license

HTTP Request

POST licenseit.pro:5001/api/License/Active

BODY Parameters
License NumberLicense key you wish to activate
Identity NumberThis is your personal key to identify the license within your app, feel free to name it as you wish. You can use this to allow specific licenses to access specific functions within your app.
Application HashThis is your application secret in which the license is registered. The secret key can be found in the main application view.
        When license is activated:
        {{"\{"}}
            "isActive": true,
            "assignedVersion": 1,
            "expiration": "2018-04-12T17:17:48",
            "description": "License Active",
        {{"\}"}}
